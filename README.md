# 📚 Book Price Analysis using Web Scraping & EDA

## Overview

This project demonstrates an end-to-end data workflow by collecting book data from the **Books to Scrape** website using Python, storing the scraped information in a CSV file, and performing Exploratory Data Analysis (EDA) to uncover pricing insights.

## Features

* Web scraping using Requests and BeautifulSoup
* Data storage using Pandas
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Price distribution visualization
* Top 10 most expensive books analysis
* Average and minimum price calculation

## Tech Stack

* Python
* Requests
* BeautifulSoup
* Pandas
* Matplotlib
* Seaborn

## Project Workflow

1. Scrape book titles and prices from the website.
2. Store the extracted data in a CSV file.
3. Load the dataset using Pandas.
4. Clean and preprocess the data.
5. Perform statistical analysis.
6. Create visualizations.
7. Generate insights from the dataset.

## Files

* `scraper.ipynb` – Web scraping notebook
* `EDA.ipynb` – Exploratory Data Analysis notebook
* `books_data.csv` – Scraped dataset

## Key Insights

* Extracted book titles and prices from a live website.
* Cleaned price values for analysis.
* Identified the top 10 most expensive books.
* Calculated the average book price.
* Found the least expensive book.
* Visualized the distribution of book prices.
